O Git Bash é um terminal extendido para otimizar o uso do Git.

Comandos básicos.

-Listar as pastas: dir

-Criar pastas/arquivos: mkdir

-Deletar pastas/arquivos:del- deleta arquivos/ remover pastas = rmdir nome da pasta /S /Q.

cd .. =volta a pasta.
limpar a tela = cls.

TAB= autocompleta uma palavra.

seta para cima = permite navegar no terminal sem ter que digitar.

git config --global user.name= para configurar o nome de usuário.
git config --global user.email=para configurar o email.
git config --list= traz uma lista de informações do git.

git checkout= retorna o arquivo para antes da edição.
git add= adiciona um arquivo
